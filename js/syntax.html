Yelp uses jQuery.Syntax (currently version 3.0) for syntax highlighting. jquery.Syntax has its own naming conventions for syntaxes. Mallard uses MIME types. DocBook has a free-form language attribute. This page shows the mapping. We try to use MIME types found in the XDG mime types package. For DocBook, we try to be compatible with language names used by docbook-style-xsl.
